<h1> a <h6>

<h1> a <h6> - representam os níveis de título de seção, sendo o <h1> é o nível mais alto e o <h6>é o elemento mais baixo.

<p>

<p> - tag que demarca um parágrafo

Voltar

<mark>

<mark> - representa um trecho de destaque de um texto, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.

Voltar

<small>

<small> - é um elemento usado para comentar e mostrar em uma fonte menor, direitos de autoral independente do estilo apresentado. Por padrão o texto marcado essa tag será menor.

Voltar

<i>

<i> - representa uma parte do texto que é destacado do restante por algum motivo, como termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Seu uso mostra o texto geralmente em itálico.

Voltar

<u>

<u> - Este elmemto representa uma span em linha, que é renderizada indicando uma anotação não-textual. Por padrão é uma linha solida simples, mas pdoe ter alterada usando o CSS.

Voltar

<strong>

<strong> - elemento que destaca a importancia do texto, titpicamente em negrito.

Voltar

<ol>

<ol> - repreenta uma lista de elementos ordenados. Como característica a ordenação é feita por numeral, letras, números romanos ou outro marcador simbólico. Esses marcadores podem ser alterados na folha de estilos CSS.

Voltar

<ul>

<ul> - repreenta uma lista de elementos não ordenados, sem uma ordem rígida. Como característica a ordenação é feita marcadores simbólico como cículo, quadrado, seta entre outros. Esses marcadores podem ser alterados na folha de estilos CSS.

Voltar

<li>

<li> - é usado para representar um item que faz parte de uma lista. Este item deve estar contido em um elemento pai: uma lista ordenada <ol> ou uma lista não ordenada <ul>. Quando for uma lista ordenda temos um contador ascendente, quando for uma lista não ordenada há um marcador como ponto ou traço.

Voltar

<a>

<a> - é o elemento de ancoragem do html, usado com o atributo href que cria um hiperlink nas páginas web, arquivos, endereços de e-mail ou mesmo endereços na mesma página ou URL.

Voltar

<hr>

<hr> - representa uma quebra temática entre elementos de nível de paragráfo. Nas versões anteriores do HTML, representava uma linha horizontal. Pode continuar sendo exibita uma linha horizontal nos navagaores, mas agora definida em termos semânticos, em vez de termos de apresentação.

Voltar

<sub>

<sub> -elemento que renderiza o texto em subescrito, geralmente usado para anotações no texto ou notação matemática mostrando o texto menor dessa forma.

Voltar

<sup>

<sup> - elemento que renderiza o texto em sobreescrito, geralmente usado para anotações no texto ou notação matemática mostrando o texto menor dessa forma.

Voltar

<blockquote>

<blockquote> - indica que o texto incluído é uma citação longa. Normalmente visto com um recuo. A URL para a fonte da citação pode ser dada usando o atributo cite e a representação do texto da fonte pode ser dada usando o <code> elemento.

Voltar

<font>

<font> - este elemento define o tamanho da fonte, porém foi descontinuado, sendo recomendado usar CSS Fonts para alterar as propriedades do estilo do texto.

Voltar

<del>

<del> - representa uma parte do texto que foi excluída de um documento. Este elemento é renderizado pelos navegadores com uma lintra entre o texto.

Voltar

<abbr>

<abbr> - elemento de abrviação, podendo oferecer uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apena ela.
